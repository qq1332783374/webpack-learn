### 1. 初始化项目
直接简单粗暴：

```
npm init -y
```

执行完命令之后会自动生成一个 `package.json` 文件，该文件定义了这个项目所需要的各种模块，以及项目的配置信息（比如名称、版本、许可证等元数据）。`npm install` 命令根据这个配置文件，自动下载所需的模块，也就是配置项目所需的运行和开发环境。
